The following panel is used to activate the anomalies.
In programmed mode anomalies need to be given a time to operate in. Anomaly requests follow the same pattern of simulation date fields.
Once the anomaly has been programmed press submit to inject it into the simulation.
 In real time mode anomalies can be activated at any time.
Press the button to activate the anomaly after the simulation has started.
